## The Core Insight
The plugin is the free acquisition channel. The token intelligence is the product.

## Pricing Tiers

### Free (Growth Engine)
- Unlimited rebinds on a single Figma file
- Single-player mode (one user, one file at a time)
- Full color, text style, font, spacing, effect rebinding
- "Restored by Token Rebinder" comment in Figma version history
- Learning mode included

### Pro — $12/month per editor ($99/year)
- Unlimited files
- Team token library sync (shared learned mappings)
- Batch rebinding (multi-page, one click)
- No version history watermark
- Token Health Score + shareable score card
- Export token mapping as JSON

### Team — $29/month per editor (min 3 seats)
- Everything in Pro
- Token Rulebook (explicit override rules)
- Cross-file consistency scoring
- Figma webhook integration (auto-rebind)
- API access (Phase 3 seed)
- Audit log

## The Genius Play: Token Intelligence Graph

Every rebind operation learns a mapping: hex X in context Y → semantic token Z.
Across thousands of users, this builds a universal token intelligence graph.

1. Better rebinding for everyone (network effect — user 10,000 gets better results than user 1)
2. Foundation for Token Bridge API ("given any CSS, tell me what design tokens it should use")
3. Acquisition bait for Figma (or defensible standalone position)

Free tier is generous because every free user trains the model.
They are not freeloaders — they are contributors to the intelligence layer.
